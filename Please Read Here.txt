================================================================================
OSINT MANAGER - README
================================================================================

A command-line program to manage OSINT data and perform searches using several 
popular tools.

================================================================================
FEATURES
================================================================================

Database Management
  - Create and manage simple JSON-backed databases of records
  - Persist OSINT research data for future reference

Email Analysis Tools
  - Holehe email enumeration (fully integrated - requires holehe package)
  - Search email addresses across 100+ platforms
  - HaveIBeenPwned breach lookup - Check if emails appear in known breaches
  - Email verification - Check email format and MX records for deliverability
  - Batch email lookups - Check multiple emails simultaneously

Domain & Network Intelligence
  - WHOIS domain lookup - Get registrant and technical information
  - DNS lookup (A, MX, NS records; install dnspython for full functionality)
  - Reverse IP lookup - Find domains/services hosted on an IP
  - Reverse DNS lookup - Get hostname from IP address
  - SSL certificate lookup - Extract certificate information from domains
  - Subdomain enumeration - Find hidden subdomains using CT logs and DNS brute force
  - Email finder by domain - Discover emails associated with company domains

People & Username Intelligence
  - Person search & OSINT - Multiple research methods:
    * Search by name across social platforms
    * Search by username with automatic profile verification
    * Search by phone number (with links to lookup services)
    * Search by email to find social profiles
    * Reverse phone lookup tools
  - Username availability checker - Check if username exists on 15+ platforms
    (Twitter, Instagram, GitHub, Reddit, YouTube, Twitch, TikTok, Discord, etc.)

Location & IP Analysis
  - IP geolocation via free ip-api.com service
  - Find geographic location and ISP information

Social Media & Web Intelligence
  - Social Media Lookback - Access whopostedwhat.com for historical posts
  - Social media profiling tools

Advanced Tools & Batch Operations
  - Advanced search tools submenu (Reverse IP, SSL, Email verification, etc.)
  - Batch operations menu:
    * Batch email breach lookup
    * Generate comprehensive OSINT reports
    * Export data to CSV/JSON formats
  - Report generation - Create detailed OSINT reports from database
  - Data export - Extract all records for external analysis

Configuration & API Management
  - Configuration menu for storing API keys (HIBP, Shodan, etc.)
  - Manage credentials securely in config.json

================================================================================
INSTALLATION
================================================================================

Dependencies are listed in requirements.txt. To install or update them, run the appropriate script for your OS – these tools now upgrade existing packages and automatically launch the program when finished.

Windows (PowerShell or CMD):
  install.bat

Unix / macOS:
  chmod +x install.sh
  ./install.sh

Running the installer will install or upgrade all packages listed in requirements.txt and then start `main.py`. You can rerun the script later to refresh dependencies.

The scripts install packages using pip --user. You may need to add the Python 
Scripts folder (Windows) or ~/.local/bin (Linux/macOS) to your PATH so that 
commands like holehe work globally.

================================================================================
USAGE
================================================================================

Run the main program using:
  python main.py

MAIN MENU OPTIONS:

1. Database Management - Create, delete, and manage JSON databases
2. Email Search (Holehe) - Search email addresses across 100+ platforms
3. Email Search (HaveIBeenPwned) - Check if emails appear in known breaches
4. Domain Lookup (WHOIS) - Get WHOIS information for domains
5. DNS Lookup - Query A, MX, and NS records
6. IP Geolocation - Find the geographic location of IP addresses
7. Social Media Lookback - Access whopostedwhat.com for historical 
   social media data
8. Subdomain Enumeration - Discover hidden subdomains and enumerate potential 
   attack surfaces
9. Person Search & OSINT - Research individuals through multiple methods
10. Advanced Search Tools - Specialized tools including:
    * Reverse IP Lookup - Find services hosted on an IP
    * SSL Certificate Lookup - Extract certificate information
    * Email Verification - Check email delivery capability
    * Username Availability Checker - Check 15+ platforms for username
    * Email Finder by Domain - Find emails associated with domains
    * Reverse DNS Lookup - Get hostname from IP
11. Batch Operations - Bulk operations including:
    * Batch Email Breach Lookup - Check multiple emails at once
    * Generate OSINT Report - Create comprehensive research reports
    * Export to CSV - Extract all data for external analysis
12. Configuration - Set up API keys for enhanced functionality
13. Exit - Close the application

Navigate the menus to manage databases or perform OSINT queries. When 
performing scans, you can save positive results to a database for later review.

================================================================================
API KEYS AND CONFIGURATION
================================================================================

For Have I Been Pwned lookups you can set the HIBP_API_KEY environment variable 
to avoid rate limits. You can also manage API keys through the Configuration 
menu within the application for Shodan and other services.

================================================================================
EXTENDING THE TOOL
================================================================================

Additional OSINT integrations can be added by implementing new methods in 
OSINTManager and updating the menu structure. The modular design makes it easy 
to plug in new lookup functions.
